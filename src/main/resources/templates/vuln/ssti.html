What is SSTI?

Server-Side Template Injection (SSTI) occurs when an attacker is able to use native template syntax to inject a malicious payload into a template, which is then executed server-side.

In this case, the application uses **Spring Expression Language (SpEL)** to process parts of the grievance. If the application doesn't properly sanitize the input, an attacker can execute arbitrary Java code.
